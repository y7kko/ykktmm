Guia rpido para o ykktmm

O mdulo tem objetivo de abstrair a maior parte da matemtica envolvida no mtodo das matrizes de transferncia. Tornando a prototipagem de materiais um joguinho de lego

In[]:
import numpy as np
import matplotlib.pyplot as plt

#importando o mdulo
import ykktmm as tmm

O mdulo composto de duas classes principais: a classe material e a classe layer.

A Classe Layer

layer representa uma nica matriz de transferncia, podendo ter 3 tipos de associao:

  • assoc='>': Uma seo normal de duto
  • assoc='//': Uma seo de duto lateral
  • assoc='||': Uma seo de tipo barreira

Caso argumentos de nmero de onda e impedncia no sejam passados para a layer, o tmm.material assume $z_c = \rho_0 c_0$ e $k = \omega / c_0$ na inicializao da classe. Essas variveis podem ser alteradas diretamente caso haja necessidade de utilizar modelo dissipativo, exemplo:

tmm.c0 = 343 + 2j
In[]:
z_cav = 1 #Aqui seria a formula da impedncia que voc quer

camadas = [
    tmm.layer(area=1, tam=0.2),
    tmm.layer(area=1, tam=0.1, zc=z_cav, assoc="//"),
    tmm.layer(area=2, tam=0.3),
    tmm.layer(area=1, tam=0.1)
    ]

A Classe Material

material representa o conjunto de todas as matrizes de transferncia. Sendo necessrio passar como argumento uma lista contendo todas as camadas e o vetor de frequncia

In[]:
f = np.arange(20,2000) #vetor de frequencias
mat = tmm.material(layers=camadas,freq=f)

Com a classe inicializada, possvel realizar as operaes esperadas para a classe

Exemplos

Os dados obtidos atravs do TMM esto sendo comparados com as equaes do Acstica de Salas, considerando modelo de ondas planas sem dissipao

Absorvedor Membrana

Um absorvedor de membrana pode ser representado como uma matriz do tipo barreira assoc='||', e duas matrizes de fludo equivalente assoc='>'.

In[]:
import pandas as pd
freq = np.arange(20,4E3,1E-2) #Vetor de frequencias

#Plot dos dados de Acustica de Salas
benchmark = pd.read_csv(r"benchmark\membrana\membrana_0_data.csv").to_numpy()
benchmark=np.append(benchmark,0) #por algum motivo exportou um frame a menos
plt.plot(freq,benchmark,label="Salas",linewidth=5)


#TMM
zp,kp = tmm.utils.delany_basley(freq,sigma=10440)
m = 485* 3E-3 #m'' = rho_m * t_m
zplaca = 1j*(2*np.pi*freq)*m #z = jwm''

mat = tmm.material(layers=[
    tmm.layer(area=1, zc=zplaca, assoc="||"),
    tmm.layer(area=1, tam=15E-2*0.2),
    tmm.layer(area=1, tam=0.8*15E-2, zc=zp, kc=kp)
    ],freq=freq)

zs = mat.solveZs() #
plt.plot(freq,tmm.utils.absort(zs),label="tmm")

#Configurando o plot
plt.xlabel("freq [Hz]")
plt.grid()
plt.xscale("log")
plt.ylim([0,1])
plt.legend()
tmm.utils.plot_cuxticks(); #muda as marcaes do eixo x
